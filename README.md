## Technologies Used
- **React Router**: For routing and remote state management
- **Tailwindcss**: for styling
- **Redux**: For state management

## Running locally

1. Clone the repo

```
git clone https://github.com/Tengzin/Forkify.git
```

2. Install dependencies

```
npm install
```

3. Start locally

```
npm start
```
